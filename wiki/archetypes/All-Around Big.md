# All-Around Big

The modern franchise big. Scores from multiple areas, anchors the defensive glass, protects the rim, and initiates offense from the high post. Wing skills layered onto a big baseline — rebounding and defensive presence are not optional.

## Anchor roster (by composite, descending)

```dataviewjs
const archetype = dv.current().archetype
const roster = dv.pages('"wiki/players"')
  .where(x => x.archetype === archetype)
  .sort(x => -x.composite)
dv.table(
  ["Composite", "Tier", "Player", "Status"],
  roster.map(x => [x.composite.toFixed(2), `T${x.tier}`, x.file.link, x.status ?? ""])
)
```
