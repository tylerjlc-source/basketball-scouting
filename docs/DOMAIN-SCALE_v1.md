# DOMAIN SCALE — v1

**Purpose:** Provides the band scale for assigning DOMAIN scores in Skill 3, parallel to how POSITION_SCALE_{GUARDS,WINGS,BIGS}_v1.md provides the band scale for assigning SUB-DOMAIN scores in Skill 2. Mechanic mirrors sub-domain scoring (R10): rank against the position peer group on the integrated capability the domain represents, then band-match against the descriptor. Domain scores are band-matched assignments, not arithmetic derivations from sub-domain scores.

**Loaded by:** Skill 3 (every evaluation) — full document.

**How to use this scale:** Each row defines a band of 5 valid scores (e.g., 8.5–8.9). The descriptor anchors the band median (e.g., 8.7). Score at the band median when the player's integrated profile matches the descriptor exactly. Score at the band floor when the player just meets the bar; dial toward the band ceiling when the player exceeds the descriptor and approaches the next-tier descriptor. Read the role-relevant sub-domains for the domain (excluding archetype structural zeros per DOMAIN-SCORE-ROLE-RELEVANCE.md), then identify which band the integrated profile fits.

**Synthesis sentence (mandatory per Skill 3):** Every domain score must be accompanied by one synthesis sentence naming the sub-domains that anchored the band-match AND the constraining sub-domain (if any) that pulled within the band. The lowest sub-domain remains visible alongside the band assignment.

**Reporting precision:** Domain scores reported X.X. R14 precision discipline applies — high-confidence band assignments use full .1 resolution within the band; low-confidence band assignments land at band floor only (.0 or .5).

**Position group selection:** Determined by archetype positional group, not listed position. Any player assigned a Guard archetype is scored against the Guards table; Wing archetypes against Wings; Big archetypes against Bigs. Domain 8 (IQ/Motor) is scored against the full NBA population regardless of position group, mirroring its all-universal sub-domain composition.

## Domain 1 — Finishing

**Sub-domains:** #1 (At-basket finishing), #2 (Contact finishing / foul draw), #3 (Post offense)

**Domain expression by position group:**
- **Guards:** Integrated interior scoring against guard length. At-rim conversion plus contact-draw skill drive the domain; post offense is structurally rare and contributes only when present. Score-First / All-Around primary-creator archetypes cluster highest; PG/Pure Playmaker and shooting-specialist archetypes cluster lower.
- **Wings:** Three-dimensional interior identity — rim, contact, and post all routinely contribute. Modern Four / Score-First Wing / All-Around Wing cluster highest; 3-and-D / Defensive Specialist / Shooting Specialist cluster lower.
- **Bigs:** Domain ceiling lives here. Interior dominance is the historical big identity. All-Around Big / Versatile Scoring Big / Energy Big cluster highest; Switchable Big / Rim Protector / Stretch Big cluster lower.

## Domain 5 — Defense

**Sub-domains:** #15 (On-ball pressure), #16 (Help defense), #17 (Rim protection), #18 (Post defense)

**Domain expression by position group:**
- **Guards:** Perimeter defense focus. On-ball pressure and help defense drive the domain; rim and post are structurally rare and contribute only when present. Defensive Specialist / 3-and-D Guard cluster highest; offensive-first archetypes cluster lower.
- **Wings:** Multi-position defensive coverage (positions 2–4). All four sub-domains routinely contribute. Defensive Specialist / Defensive-Engine Playmaker / 3-and-D Wing / All-Around Wing cluster highest; Shooting Specialist / Score-First Wing cluster lower.
